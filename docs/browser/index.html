ABOUT ME


I'm Vaibhav, a 26-year-old Full Stack Developer with over 3 years of experience based in Nagpur, India. My journey into programming began shortly after graduation when I started self-learning web development through online resources such as YouTube tutorials and blog posts.

In early 2022, I joined GIF Technologies as an Assistant Executive Developer, where I gained hands-on experience with Angular and AngularJS frameworks.

At Com1 Communication Technologies Pvt. Ltd, I led the migration of our application from Angular version 10 to version 14. This project significantly enhanced the application's performance, security, and alignment with modern web standards.

One of my notable projects is a web application designed to send personalized emails containing QR codes. It involves a backend built with Node.js and Express.js, which handles QR code generation, email sending using Nodemailer, and logging email data into MongoDB. The frontend, built with Angular, enables users to input multiple email addresses, validate them, and send requests to the backend. The application is designed to streamline marketing campaigns, event invitations, or attendance tracking through dynamic and personalized QR codes. You can access this project from my GitHub repository for further details and implementation.

As a freelancer, I also worked on various web development projects, gaining valuable experience in building dynamic web applications.

I have worked on multiple projects, developing dynamic web applications and consistently aiming to learn more, do more, and excel. I am dedicated, curious, passionate, committed, and a quick learner!
